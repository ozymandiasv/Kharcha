☁️ GitHub Gist Sync

Store all your data — entries, categories, stores, and icons — in a private GitHub Gist. Pull to load from cloud, push to save your latest data.

Needs gist scope. Stored locally only, never sent anywhere except GitHub.
ℹ️ How to set up
  1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Generate new token, select only gist scope
  3. Paste the token above and click Push to Gist
  4. The Gist ID will be saved automatically for future syncs
  5. On a new device: enter your token + Gist ID, then click Pull from Gist
